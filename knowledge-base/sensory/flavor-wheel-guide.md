# SCA Coffee Taster's Flavor Wheel Guide

The Coffee Taster's Flavor Wheel is the industry standard tool for describing coffee flavors, developed by the Specialty Coffee Association (SCA) in collaboration with World Coffee Research (WCR).

## Overview

- **Original Version**: 1995
- **Current Version**: 2016 (major revision)
- **Foundation**: World Coffee Research Sensory Lexicon (110 attributes)
- **Purpose**: Standardized vocabulary for coffee flavor description

## How to Use the Flavor Wheel

### Start from the Center
1. Begin at the center with broad categories
2. Work outward to more specific descriptors
3. The further out you go, the more precise the description

### Nine Primary Categories

| Category | Description | Common Descriptors |
|----------|-------------|-------------------|
| **Fruity** | Fruit-like flavors | Berry, citrus, dried fruit, other fruit |
| **Sour/Fermented** | Acidic, fermented notes | Sour, alcohol, fermented |
| **Green/Vegetative** | Plant-like, unripe | Olive oil, raw, green, beany |
| **Other** | Miscellaneous | Papery, chemical, pipe tobacco |
| **Roasted** | Roast-derived flavors | Cereal, burnt, tobacco |
| **Spices** | Spice notes | Pungent, pepper, brown spice |
| **Nutty/Cocoa** | Nut and chocolate | Nutty, cocoa |
| **Sweet** | Sweetness types | Brown sugar, vanilla, overall sweet |
| **Floral** | Flower-like aromas | Black tea, floral |

## Intensity Scale

The WCR Sensory Lexicon uses a 0-15 intensity scale:
- **0-3**: Very low intensity
- **4-6**: Low to moderate
- **7-9**: Moderate to high
- **10-12**: High intensity
- **13-15**: Very high intensity

## Using Flavor Descriptors in Practice

### For Cupping
1. Smell dry grounds - note fragrance
2. Smell wet grounds after adding water - note aroma
3. Taste and identify broad categories first
4. Refine to specific descriptors
5. Note intensity levels

### Common Origin Profiles

| Origin | Typical Descriptors |
|--------|-------------------|
| Ethiopia (Yirgacheffe) | Floral, bergamot, blueberry, citrus |
| Kenya | Blackcurrant, tomato, citric acid, wine |
| Colombia | Caramel, nuts, red apple, balanced |
| Brazil | Chocolate, nuts, low acidity, heavy body |
| Guatemala | Chocolate, spice, apple, balanced |
| Costa Rica | Honey, citrus, clean, bright |
| Panama (Geisha) | Jasmine, bergamot, peach, tea-like |
